# Fryrvo Discs 
**Add more Music Discs to your game**
<br>
**_This mod adds custom music discs tracks By Fryrvo to the game_** 

### All Current Music Discs
![All Current Music Discs](https://cdn.modrinth.com/data/cached_images/407dff96f34cb775a34c4bc6e4b098d5a5d31319.png)

### Detail
* **Mod Loader:** Fabric
* **Latest Versions:** 1.21.8–1.21.11
* **Dependencies:** Fabric API


### How to Obtain
* **Survival:** Currently UnObtainable in Survival
* **Creative:** "Tools & Utilities" creative tab.
